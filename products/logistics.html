01 / Задача

Транспортные компании обязаны вести ЭПЛ/ЭТрН в электронном виде через ФГИС ЭПД. Существующие решения требуют мобильных приложений для каждого участника, отдельных учёток, обучения, КЭП на водителя.

Система bit делает то же самое через корпоративный мессенджер МАКС: водитель, механик, медик, представитель контрагента работают в чатах без приложений и логинов. Внедрение измеряется днями, а не месяцами.

02 / Решение
  • Inline-кнопки в чате МАКС вместо мобильного приложения
  • Подпись в мессенджере: ПЭП через Контур, УНЭП через Госключ
  • Полная интеграция с Контур.Логистика API
  • Сдача в ФГИС ЭПД автоматически в формате ФНС 5.01
  • 6 титулов ЭПЛ согласно приказу ЕД-7-26/116@
  • Live-monitoring смен для диспетчера
